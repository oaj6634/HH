# HH

Client - https://github.com/kimkyung1234/HH-Client

자기의 애완동물을 소개하는 글 쓰기 앱
***
<h1>배포환경</h1>
AWS EC2를 통한 배포

***
<h1>사용 스킬</h1>
<ul>
    <li>JAVA11</li>
    <li>Spring Boot</li>
    <li>Spring Security</li>
    <li>AWS</li>
    <li>Mysql</li>
</ul>

***
<h1>구현 기능</h1>
<ul>
    <li>계정 관련</li>
        <ul>
            <li>회원가입 기능</li>
            <li>JWT를 활용한 로그인 기능</li>
            <li>프로필 수정, 등록, 조회 기능</li>
        </ul>
    <li>게시물</li>
    <ul>
        <li>게시물 등록, 조회 기능</li>
        <li>게시물 5개 단위로 페이징</li>
        <li>특정 유저가 올린 게시물 조회 기능</li>
    </ul>
</ul>

***
